
# URl Shortener

 It takes a long URL and turns it into a short link that is easier to share
 

## Run Locally

Clone the project

```bash
  git clone https://github.com/Gauravrandheer/Url_shortener.git
```

Install dependencies

```bash
  npm install
```
Start the server

```bash
  npm index.js
```
